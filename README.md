# 🍔 Burger King Homepage Clone

A static front-end clone of the Burger King website built using only **HTML5** and **CSS3**.

This project recreates the visual appearance and layout of the Burger King homepage, including the navigation bar, menu section, promotional banners, explore section, and footer. The website is completely static and focuses on front-end design implementation without using JavaScript or external frameworks.

---

## 📌 Features

* Responsive homepage layout
* Burger King inspired design
* Navigation bar with delivery and dine-in options
* Store location search section
* Menu categories section
* Promotional banners
* Explore section
* Footer with company information
* Pure HTML and CSS implementation

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## 🚫 Limitations

This is a static clone project and does not include:

* JavaScript functionality
* Online ordering system
* Authentication/Login system
* Cart functionality
* API integration
* Backend services
* Database connectivity
* Animations or dynamic interactions
